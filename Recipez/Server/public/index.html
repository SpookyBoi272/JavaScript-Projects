Recipiz API v1

Following API are availabe as of now:

1. Get page

This endpoint provides a list of complete recipes for a Page. Each page contains 20 recipes.

Usage:

/v1/getPg/pageNo

2. Search Ingredients

This endpoint provides a list of search result of IngredientsNER. Each list contains 5 or less Ingredients.

Usage:

/v1/search/ner/?q=query
